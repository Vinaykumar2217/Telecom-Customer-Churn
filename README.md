Telecom Customer Churn Project

Project Description:
This project focuses on analyzing customer behavior and identifying factors that contribute to customer churn in the telecommunication industry. Customer churn refers to customers discontinuing services and switching to other providers. The project applies data preprocessing, exploratory data analysis, statistical techniques, feature engineering, dimensionality reduction, and machine learning algorithms to understand customer patterns and predict churn.

Objectives:
• Perform complete data acquisition and preprocessing
• Handle missing values and duplicates
• Detect and treat outliers
• Perform encoding and feature scaling
• Conduct univariate, bivariate, and multivariate analysis
• Apply statistical analysis techniques
• Perform feature engineering and PCA
• Build predictive machine learning models

Dataset Information:
Dataset Name: Telecom Customer Churn Dataset
Source: Kaggle
Records: 7043
Features: 21
Target Variable: Churn
